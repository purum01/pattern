데코레이터 패턴(Decorator Pattern) 적용하여 출력 장식 구현하기

📋 실습 목표

하나의 문자열을 다양한 방식으로 장식하여 출력하는 프로그램을 데코레이터 패턴으로 구현합니다.
각 장식 요소를 독립된 클래스로 정의하고, 중첩 가능하도록 설계하는 것이 목표입니다.

장식을 상속으로 구현하는 대신, Display 인터페이스와 Border 클래스를 통해 동적으로 조합하세요.

🛠 구현 지침

  • Display: 출력 대상의 기본 인터페이스 (getRows(), getColumns(), getRowText())
  • StringDisplay: 출력할 문자열을 하나 가지고 있음
  • Border: Display를 상속받고, Display를 멤버로 가짐 (장식의 부모 클래스)
  • SideBorder: 좌우에 문자 추가
  • FullBorder: 상하좌우 모두 테두리 추가
  • UpDownBorder: 상하에만 문자로 장식
  • 모든 장식은 Display 인터페이스로 다시 포장되어, 조합이 가능해야 합니다.

🖥 실행 예시


  ~~~~~~~~~~~~~~~
  +-------------+
  |#Hello, world.#|
  +-------------+
  ~~~~~~~~~~~~~~~

🗂 제공 파일 (리팩토링 전)

FramePrinter.java


package decorator;

public class FramePrinter {
    private String text;

    public FramePrinter(String text) {
        this.text = text;
    }

    public void printWithDecoration() {
        String line = "+-------------+";
        System.out.println("~~~~~~~~~~~~~~~");
        System.out.println(line);
        System.out.println("|#" + text + "#|");
        System.out.println(line);
        System.out.println("~~~~~~~~~~~~~~~");
    }
}

Main.java


package decorator;

public class Main {
    public static void main(String[] args) {
        FramePrinter printer = new FramePrinter("Hello, world.");
        printer.printWithDecoration();
    }
}
