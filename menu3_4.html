상태 패턴(State Pattern) 적용하여 리팩토링하기

📋 실습 목표

주어진 코드는 시간대(주간/야간)에 따라 동작이 달라지는 보안 시스템입니다.
조건문을 사용한 방식을 상태 패턴(State Pattern)으로 리팩토링하세요.

조건 분기(if) 대신 상태 객체에 행동을 위임하고,
시각에 따라 자동으로 상태(State)를 변경하도록 구현하세요.

🛠 리팩토링 지침

  • State 인터페이스를 정의하고 행동을 선언합니다.
  • DayState, NightState 클래스를 구현하여 시간에 따라 다르게 행동하도록 합니다.
  • Context 인터페이스를 통해 상태 변경 및 기록/출력을 처리하도록 합니다.
  • SafeFrame 클래스는 Context 역할로 현재 상태에게 요청을 위임합니다.

📂 리팩토링 후 파일 구조

🗂 제공 파일 (리팩토링 전)

아래 코드에서 시간 조건에 따른 if 분기들을 제거하고, 상태 패턴을 적용하세요.

SafeFrame.java

package state;
  
  import java.awt.*;
  import java.awt.event.*;
  
  public class SafeFrame extends Frame {
      private TextField textClock = new TextField(60);
      private TextArea textScreen = new TextArea(10, 60);
      private Button buttonUse = new Button("금고사용");
      private Button buttonAlarm = new Button("비상벨");
      private Button buttonPhone = new Button("일반 통화");
      private Button buttonExit = new Button("종료");
  
      public SafeFrame(String title) {
          super(title);
          setBackground(Color.lightGray);
          setLayout(new BorderLayout());
  
          add(textClock, BorderLayout.NORTH);
          textClock.setEditable(false);
  
          add(textScreen, BorderLayout.CENTER);
          textScreen.setEditable(false);
  
          Panel panel = new Panel();
          panel.add(buttonUse);
          panel.add(buttonAlarm);
          panel.add(buttonPhone);
          panel.add(buttonExit);
          add(panel, BorderLayout.SOUTH);
  
          pack();
          setVisible(true);
  
          buttonUse.addActionListener(e -> {
              int hour = getCurrentHour();
              if (hour < 9 || hour >= 17) {
                  textScreen.append("야간 금고 사용 기록\n");
              } else {
                  textScreen.append("주간 금고 사용 기록\n");
              }
          });
  
          buttonAlarm.addActionListener(e -> textScreen.append("비상벨 작동\n"));
          buttonPhone.addActionListener(e -> textScreen.append("전화 연결\n"));
          buttonExit.addActionListener(e -> System.exit(0));
      }
  
      public void setClock(int hour) {
          textClock.setText("현재 시각은 " + (hour < 10 ? "0" + hour : hour) + ":00");
      }
  
      private int getCurrentHour() {
          return Integer.parseInt(textClock.getText().replaceAll("\\D", ""));
      }
  }

Main.java

package state;
  
  public class Main {
      public static void main(String[] args) {
          SafeFrame frame = new SafeFrame("State Sample");
          for (int hour = 0; hour < 24; hour++) {
              frame.setClock(hour);
              try {
                  Thread.sleep(1000);
              } catch (InterruptedException e) {}
          }
      }
  }
